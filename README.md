# 🛒 Lista de Compras

Projeto desenvolvido como desafio técnico utilizando **HTML, CSS e JavaScript**, com foco em manipulação do DOM e interatividade.

## Funcionalidades

- Adicionar novos itens à lista
- Editar nome e quantidade dos itens
- Remover itens da lista
- Atualização automática da quantidade de itens
- Layout responsivo

## Acesse a aplicação:

**https://claudiabeathriz.github.io/tarefa-dom-in-junior/**

## Conceitos aplicados

- Manipulação do DOM
- Eventos (`addEventListener`)
- Arrays e objetos
- Atualização dinâmica da interface
- Estruturas condicionais e funções
- Organização de código em JavaScript
